# Ena Syntax Highlighter

This is the VS code syntax highlighter extension for the Ena language.

## Features

Highlight all the main programming language features:
- Function names
- Function parameters
- Variables
- Comments
- Strings
- Numbers
- Operators
- Keywords (Georgian and English)
